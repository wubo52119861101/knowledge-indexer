## 说明
- 一期编码实现优先交付可运行骨架与核心链路，默认使用内存仓储与确定性哈希向量，便于先跑通 API 与领域流程。
- PostgreSQL/pgvector、Redis、MinIO、CocoIndex 流程与 Postgres 数据源均已预留扩展点，后续可在不破坏接口契约的前提下替换实现。
